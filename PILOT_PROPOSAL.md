# Pilot Proposal — GeoSpectra ScanGuard

## Objective

Validate spectral 3D defect detection on real industrial scans from partner facilities.

## What We Provide

1. Free on-site software deployment
2. Real-time anomaly detection dashboard
3. Detailed inspection reports per scan
4. Full data confidentiality (no cloud upload)

## What We Need From Partner

1. 1–2 golden reference scans per part type
2. 10–20 production scans (mix of good and defective)
3. Ground-truth labels for defective scans (if available)
4. 15-minute feedback session per week

## Duration

4 weeks per part type.

## Success Criteria

- <5% false positive rate on known-good scans
- >80% detection rate on known-defective scans
- Processing time <3 seconds per scan

## Deliverables

1. Weekly performance reports
2. Final validation report with honest limitations
3. Source code (MIT license)
4. Technical documentation
